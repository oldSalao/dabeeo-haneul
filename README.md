# 실행 방법.

1. repository를 clone 합니다.

   ```
   git clone https://github.com/oldSalao/dabeeo-haneul.git
   ```

2. clone된 프로젝트 폴더 내에서 터미널을 통해 yarn 명령을 수행합니다.

   ```
   yarn
   ```

3. 이어서 yarn start를 수행합니다.

   ```
   yarn start
   ```
